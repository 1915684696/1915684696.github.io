jQuery等在myeclipse中报错的问题

从官方下载的jquery.js在myeclipse始终用个大大的红叉，看着很不爽，解决方法：

  1. 选中报错的jquery文件
  2. 右键选择 MyEclipse–>Exclude From Validation
  3. 左面点击 -> Excluded resources
  4. 找到jquery.js -> 打上钩 -> apply
