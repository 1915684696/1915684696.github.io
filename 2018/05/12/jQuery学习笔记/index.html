jQuery学习笔记

jQuery是JavaScript一个比较火的框架，我在这里暂时是它的一些核心的部分，日后会边学边加。

js对象和jquery对象的区别

jquery就是js兑现的new object生成的普通对象，然后在jquery对象中加入了很多的方法，经过不断地完善，就形成了jquery对象。

<script>
function $(){
    obj = new Object();
    obj.username='user123';
    obj.say=function(){
        alert('my name is :'+this.username);
        return obj;
    }
    obj.eat=function(){
        alert('I am eating');
    }
    return obj;
}

$().say().eat();    //链式开发结构

</script>

jquery和js对象的方法能不能共用？
答：不能，jquery对象和js对象原本就是两个不同的对象，方法自然不能共用
jquery对象和js对象能不能互换？
答：能，
dom对象->jquery对象:$(dom)

<body>
<img src="a.jpg" id="imgobj">
</body>
<script src="jquery.js"></script>
<script>
imgobj = document.getElementById('imgobj');
/*imgobj.onclick=function(){
    this.src='b.jpg';
}*/


$(imgobj).click(function(){
    //this.src='b.jpg';        //this代表dom对象
    $(this).attr({'src':'b.jpg'});        //setAtrrbute()
});

</script>

val = $('h1')[0].outerHTML;    //jquery没有outerHTML,[1]则是第二个h1标签

jquery核心方法

each();
size();
length;
get();
get(index);
index();
data();

size()和length都是显示jquery对象中dom对象的个数，注意length这里是一个属性，并不是方法

<body>
<h1>aaaaaaaaaaaa</h1>
<h1>bbbbbbbbbbbb</h1>
</body>
<script>
    alert($('h1').length);    //弹出2
    alert($('h1').size());    //弹出2，注意jQuery1.8.*以后不用
</script>

get()函数返回的是jquery对象中的dom对象数组，get(index)函数获得jquery对象中的dom对象数组的第index+1个元素

    <body>
<h1>aaaaaaaaaaaaaaaaaaaa</h1>
<h1>bbbbbbbbbbbbbbbbbbbb</h1>
</body>
<script src="jquery-3.3.1.js"></script>
<script>
arr = $('h1').get();    //jquery没有outerHTML
alert(arr[1].outerHTML);//弹出框<h1>bbbbbbbbbbbbbbbbbbbb</h1>

val = $('h1').get(1).outerHTML;    //[1]其实是第二个h1标签
alert(val);                        //弹出框<h1>bbbbbbbbbbbbbbbbbbbb</h1>
</script>

each()函数是实现jquery对象遍历的一个方法。如下边的例子（单击元素显示行号）：

<body>
<h1>aaaaaaaaaaaa</h1>
<h1>aaaaaaaaaaaa</h1>
<h1>aaaaaaaaaaaa</h1>
<h1>aaaaaaaaaaaa</h1>
<h1>aaaaaaaaaaaa</h1>
</body>
<script>
$('h1').each(function(i) {
    $('h1').get()[i].setAttribute('num',i+1);
});
$('h1').click(function(){
    this.innerHTML = this.getAttribute('num');
});
</script>
//一般的情况更多的使用下边的形式,统一使用jquery对象或者dom对象，不会来回的转化
<script>
$('h1').each(function(i) {
    $(this).attr({num:i+1});
});
$('h1').click(function(){
    $(this).html($(this).attr('num'));    //html()函数类似innerHTML()函数，在这里赋值了
});
</script>

index()函数搜索匹配的元素，并返回相应元素的索引值
