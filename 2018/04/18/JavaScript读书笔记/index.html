JavaScript学习笔记

笔记参考《JavaScript实战》（西南财经大学出版社）

一、使用JavaScript

<script>XXX</script>这组标签，是用于在html页面中插入js的主要方法。它有以下几个属性：

  • charst：可选，表示src属性表示的字符集。很少用。
  • defer：可选，表示脚本可以延迟到文档完全被解析和显示之后再执行。很少用。
  • language：已废弃，原来用于代码使用的脚本语言。已不用。
  • src：可选，表示包含要执行代码的外部文件。
  • type：必需。表示代码使用的脚本语言的内容类型。type=’text/javascript’。

将JavaScript另存为.js文件

<script type="text/javascript" src="demo1.js"></script>

这时里边不能有任何代码

我们通常会将<script></script>标签放在<head>····</head>之间，有事也会放在<body>····</body>之间。

二、JavaScript数据类型

Undefined

该类型只有一个特殊的值undefined。当使用var声明变量但没有对其初始化时该值就是undefined.

Null

同样 只有一个特殊的值null。表示空对象的引用（指针）。用typeof检测会返回object。

Boolean

只有两个值true和false。

Number

包括两种数值：整形和浮点型。

支持进制，如：

var num = 070;    //八进制，56
var num = 0xA;    //十六进制，79

NaN：一个特殊的值，感觉是为了不报错而产生的，如：

var box = 0/0;    //NaN

函数isNaN()用来判断一个值是不是NaN，如：

alert(isNaN(NaN));   //true
alert(isNaN(NaN));   //false

有三个转型函数，可以把非数值转换为数值：Number()、parseInt()、parseFloat(),Number()可以用于任何数据类型，另外两个专门用于将字符串转换化数值

alert(Number(true));//1,Boolean类型的true和false分别转换为1和0
alert(Number(25);//25,数值型直接返回
alert（Number(null));//0,空对象返回0
alert(Number(undefined));//NaN,undefined返回NaN

对于字符串而言，

  • 只包含数值的会直接转化为十进制
  • 只包含浮点数值得字符串会直接转化为浮点数值
  • 空字符串转化为0
  • 其他的转化为NaN

String

由双引号（””）或单引号（’’）表示，同其他的编程语言有着同样的转义符号\

toString()方法可以把值转换成字符串，如：

var box = 11;
alert(typeof box.toString());

toString()方法一般不需要传递参数，但在数值转换为字符串的时候可以传递参数，如：

var box = 10;
alert(box.toString());//10,默认输出；
alert(box.toString(2));      //1010,二进制输出
alert(box.toString(8));      //12，八进制输出
alert(box.toString(10));    //10,十进制输出
alert(box.toString(16));    //a,十六进制输出

Object

跟Java相同， 对象类型，创建可以用new+要创建的对象类型的名称来创建，如：

var box = new Object();

Object()里可以任意传参数，可以传数值、字符串、布尔值等。还可以进行相应的计算。如：

var box = new Object(2);     //Object类型，值为2
var age = box + 2;           //可以和普通变量计算
alert(age);                  //输出结果，转型为Number类型4

三、流程控制语句之for…in语句

for…in语句是一中迭代语句，可以用来枚举对象的属性，如：

var box = {
  'name' : '高寒';
  'age' : 28;
  'height' : 178;
};
for(var p in box){
  alert(p);
} `

四、函数

函数的声明

JavaScript统一使用function关键字声明函数，后跟一组参数以及函数体，如：

function box(){
alert('只有函数被调用，我才会被之执行');
}

return返回值

JavaScript的函数都不需要定义返回值，是在调用后直接执行的。如：

function box(){
  return '我被返回了';
}
alert(box());

arguments对象

JavaScript函数不介意传递进来多少参数，也不会因为参数的不统一而错误。函数体内可以通过arguments对象来接收传递进来的参数。如：

//弹出框1|2
function box(){
    return arguments[0] + '|' + arguments[1];
}
alert(box(1,2,3,4,5,6));

arguments对象的length属性可以可到参数的数量。

五、对象

Object的创建

object的创建分两种，一种是new运算符，另一种是字面量表示法。（new跟Java类似）
使用new运算符，new也可省略

var box = new Object();
box.name = '宁浩';
box.age = 19;

使用字面量方式创建Object

var box = {
    name : '宁浩',        //这里注意一定是“,”符号，“；”会出错
    age : 19;
}

或者

var box ={};
box.name = '宁浩',
box.age = '19';

Object属性的输出方式

alert(box.age);
alert(box['age'];

为Object创建方法

var box = {
    run : function(){
        return '我是return';
    }
}
alert(box.run());            //调用对象中的方法

六、数组

一个复杂的数组例子

var box = [
    {                                //第一个元素是对象
    name : '宁浩',
    age : 19,
    run : function(){
        return '我是return出来的';
        }
    },
    ['张三','李四',new Object()],    //第二个元素是数组
    '杭州',                            //第三个元素是字符串
    25 + 25,                        //第四个元素是数值
    new Array(1,2,3)                //第五个元素是数组
    ];
alert(box);

数组中的几个方法

转换方法

对象或数组都具有toLocaleString()、toString()和valueOf()方法，数组会将每个值进行字符串的拼接，以逗号隔开。
使用join()方法可以使用不同的分割符

var box = ['张三','李四'];
alert(box);                        //隐式调用toString()方法，弹出框 张三，李四
alert(box.toString());            //弹出框 张三，李四
alert(box.toLocaleString());    //弹出框 张三，李四
alert(box.join('|'));            //弹出框 张三|李四

栈和队列的方法

跟数据结构中的栈相同，JavaScript的数组可以看做是栈，提供了push()方法、pop()方法和shift()方法。push()方法可以接受任意数量的参数并将它们逐个添加到数组的末尾，返回修改后的长度。pop()方法可以从数组末尾移除最后一个元素，减少数组的length值，返回移除的元素。shift()方法则是从数组的前端移除一个元素，返回移除的元素。

var box = ['张三',28];
alert(box.push('杭州'));        //3
alert(box);                    //张三,28,杭州
box.pop();
alert(box);                    //张三,28
alert(box.shift());            //张三
alert(box);                    //28

排序方法

数组中有两个排序方法：reverse()和sort()方法。reverse()逆向排序，sort()从小到大排序。
注意：这两个函数全部都是引用，数组直接被改变。

var box = [1,2,3,4,5];
alert(box.reverse());    //5,4,3,2,1
alert(box);                //5,4,3,2,1
alert(box.sort());        //1,2,3,4,5
alert(box);                //1,2,3,4,5

七、BOM

BOM也叫浏览器对象模型，它提供了很多对象，用于访问浏览器的功能。BOM缺少规范，每个浏览器提供商 又按照自己想法去扩展它，那么浏览器共有对象就成了事实的标准。所以，BOM本身是没有标准的或者还没有哪个组织去将它标准化。

window对象

BOM的核心是window，它表示浏览器的一个实例，window对象处于JavaScript结构的最顶层，对于每个打开的窗口，系统都会自动为其定义一个window对象。

window的对象属性

window对象有一系列的属性，这些属性本身也是对象。

属性 含义
closed 当窗口关闭时为真
defaultStatus 窗口底部状态栏显示的默认状态消息
document 窗口中当前显示的文档对象
frames 窗口中的框架对象数组
history 保存有窗口最近加载的URL
length 窗口中的框架数
location 当前窗口的URL
name 窗口名
offscreenBuffering 用于绘制新窗口内容并在完成后复制已存在的内容，控制屏幕更新
opener 打开当前窗口的窗口
parent 指向包含窗口的窗口（由框架使用）
screen 显示屏幕相关信息，如高度、宽度（以像素为单位）
self 指示当前窗口
status 描述由用户交互导致的状态栏的临时消息
top 包含特定窗口的最顶层窗口（由框架使用）
window 指示当前窗口，与self等效

    alert('您按了取消');

新建窗口

使用window.open()方法可以导航到一个特定的URL,也可以打开一个新的浏览器窗口。它可以接受四个参数：

  1. 要加载的URL；
  2. 窗口的名称或窗口目标；
  3. 一个特性字符串；

  4. 一个表示新页面是否取代浏览器记录中当前加载页面的布尔值。

    open(‘http://www.baidu.com');
    open(‘http://www.baidu.com','baidu');
    open(‘http://www.baidu.com','baidu','width=400,height=400,top=200,left=200,toolbar=yes');

    location对象

    location是BOM对象之一，它提供了与当前窗口中加载的文档有关的信息，还提供了一些导航功能。事实上，location对象是window对象的属性，也是document对象的属性；所以window.location和document.location等效。

    alert(location); //获取当前的URL

八、DOM基础

DOM，英文是Document Object Model,即文档对象模型。它描绘了一个层次化的节点树

DOM简单操作

我们可以通过简单的操作定位节点的方法和属性

方法 说明
getElementById() 获取特定ID元素的节点
getELementByTagName() 获取相同元素的节点列表
getElementByName() 获取相同名称的节点列表
getAttribute() 获取特定元素节点属性的值
setAttribute() 设置特定元素节点属性的值
removeAttribute() 移除特定元素节点属性

当我们获取特定元素节点之后，这个节点对象就被我们获取到了，通过这个节点对象，我们就可以访问它的的属性了。

属性 说明
tagName 获取元素节点的标签名
innerHTML 获取元素节点里的内容
id 节点id名称
title 节点的title属性
style css内联样式属性
className css元素的类

DOM节点属性

节点可以分为元素节点、属性节点和文本节点，而这些节点又有三个属性nodeName、nodeType、nodeValue。

节点类型 nodeName nodeType nodeValue
元素 元素名称 1 null
属性 属性名称 2 属性值
文本 #test 3 文本内容（不包括html）

层次节点属性

节点的层次结构可以分为父节点与子节点、兄弟节点两种。我们可以通过层次节点属性获取已知的元素节点的其他相关层次的节点。

属性 说明
childNodes 获取当前元素节点的所有子节点
firstChild 获取当前元素节点的第一个子节点
lastChild 获取当前元素节点的最后一个子节点
ownerDocument 获取当前元素节点的文档根节点，相当于document
parentNode 获取当前节点的父节点
nextSibling 获取当前节点的前一个同级节点
nextSibling 获取当前节点的后一个同级节点
attributes 获取当前元素节点的所有属性节点集合

节点的操作

DOM不仅仅可以查找节点，也可以创建节点、复制节点、插入节点、删除节点和替换节点。

方法 说明
write() 这个方法可以把任意字符串插入到文档中
createElement() 创建一个元素节点
appendChild() 将新节点追加到子节点列表的末尾
createTextNode() 创建一个文件节点
insertBefore() 将新节点插入在前面
replaceChild() 将新节点替换旧节点
cloneNode() 复制节点
removeChild() 移除节点
